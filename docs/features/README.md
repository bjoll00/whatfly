### Fly Suggestions
The app's core feature is suggesting flies based on:
- Real-time weather conditions
- Water conditions (flow, temperature, clarity)
- Time of day and season
- Location-specific data

See [FLY_SUGGESTIONS_IMPLEMENTATION.md](./FLY_SUGGESTIONS_IMPLEMENTATION.md) for details.

### Map Integration
Interactive map with location selection and real-time data fetching.

### Database
Comprehensive fly database with detailed attributes for accurate suggestions.
